# Abhinandan's Personal Portfolio 🚀

Welcome to the repository for my personal portfolio website! This site serves as a digital resume and a centralized hub to showcase my skills, projects, and academic journey as a Computer Science & Engineering student.

🔗 **Live Demo:** [my-portfolio-abhinandan.vercel.app](https://my-portfolio-abhinandan.vercel.app)

## 📖 About This Project

This portfolio was designed with a focus on clean aesthetics, readability, and modern web design principles. It features a premium dark-mode UI and is fully responsive across all devices. The site highlights my technical arsenal (Python, C++, Java, SQL), my recent internship experience, and my core projects like the Cyber Defensive Engine and Dr. Vaccine Antivirus.

## ✨ Features

* **Premium Dark Theme:** A sleek, modern aesthetic tailored for a cybersecurity and software engineering profile.
* **Fully Responsive:** Fluid layout that adapts seamlessly to desktop, tablet, and mobile screens.
* **Dynamic Resume Link:** Direct integration with Google Drive to ensure the downloadable CV is always the most up-to-date version.
* **Interactive UI Elements:** Glassmorphism card effects, smooth hover transitions, and sticky navigation.

## 🛠️ Tech Stack

This project is built using vanilla web technologies to keep it lightweight and fast:

* **HTML5** (Semantic structure)
* **CSS3** (Custom styling, CSS variables, Flexbox/Grid, Animations)
* **Google Fonts** (Inter typeface)
* **Deployment:** Vercel
